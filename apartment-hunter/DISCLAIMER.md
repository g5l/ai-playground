# Legal & Ethical Disclaimer

## Purpose

This tool is a **personal research assistant** for apartment hunting in Porto Alegre, Brazil.
It is intended solely for private, non-commercial use by an individual to aggregate publicly
available real estate listings for personal decision-making.

## Web Scraping Notice

This project may collect publicly accessible listing data from real estate portals (e.g., VivaReal,
ZAP Imóveis). Before using any scraper component:

1. **Review the Terms of Service** of each target website. Many portals explicitly prohibit
   automated data collection.
2. **Respect `robots.txt`** files. The scraper must honor crawl delays and disallowed paths.
3. **Rate limiting** is enforced via `SCRAPE_RATE_LIMIT_MS` to avoid overloading servers.
4. **Do not use this tool** to collect, store, or redistribute personal data about individuals
   (e.g., seller contacts, owner information).

## Data Handling

- All data is stored **locally** on your machine in a SQLite database.
- No data is transmitted to third parties (except the LLM API calls to Anthropic for ranking,
  which send only listing attributes — no personal information).
- Raw scraped payloads are stored transiently in `data/raw/` and may be deleted at any time.

## AI-Generated Rankings

Rankings and rationales generated by the AI (Claude via Anthropic API) are **suggestions only**
and may be inaccurate, biased, or outdated. Always verify listing information directly with
the real estate agent or portal before making any financial decisions.

## Responsible Use

- Use the scraper infrequently (e.g., once per day via cron).
- Do not attempt to bypass authentication, CAPTCHAs, or other access controls.
- If a website requests that you stop scraping, stop immediately.
- Consider reaching out to portals about official API access for your use case.
